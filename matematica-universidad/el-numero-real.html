Números reales

Conjunto de los números reales

Los números reales, que simbolizamos con R, es el conjunto de números formado por todos los números racionales, Q, más los irracionales, I.

Reales positivos: R+ = {x∈R/ x>0}

Reales negativos: R- = {x∈R/ x<0}

Reales no negativos: R+o = {x∈R/ x≥0}

Reales no positivos: R-o = {x∈R/ x≤0}

Propiedades de los números reales

  • x > y; a ∈ R => x+a > y+a
  • x > y; a > 0 => x.a > y.a
  • x > y; a < 0 => x.a < y.a
  • x > y; a > b => x+a > y+b
  • x/y ∈ Q; a/b ∈ Q; x/y > a/b => x.b > a.y

La recta real

El segmento OP se obtiene de desplazar 3 veces el segmento OI hacia el lado derecho, y al punto P se lo llama "coordenada", y se lo indica P(3).

Valor absoluto

El valor absoluto de un número x, simbolizado como |x| es un número real que

  • es igual a x, si x es positivo ó cero
  • y es el opuesto de x, si x es negativo

Distancia (d) entre puntos de una recta

Dado un punto A sobre una recta graduada, si "a" es su coordenada, entonces d(A,O)=|a|

Dados dos puntos A y B sobre una recta graduada, si a y b son sus respectivas coordenadas, entonces d(A,B)=|b-a|

La distancia entre A y B es la misma que entre B y A

El plano coordenado

Plano cartesiano, denotado R2 ya que se usan dos copias de R ubicadas de forma perpendicular (ortogonal) una de otra, formando un sistema cartesiano ortogonal (sistema de referencia plano donde los ejes son ortogonales)

Coordenadas cartesianas

En un plano cartesiano, cualquier punto se localiza mediante un par ordenado de números reales llamados coordenadas cartesianas del punto

Distancia entre puntos del plano

La distancia entre puntos del plano se obtiene usando el teorema de Pitágoras, y se calcula de la siguiente forma:

d(P1 ,P2)= [ (x2-x1)2 + (y2-y1)2 ]1/2

Trigonometría

Funciones trigonométricas básicas

Estas funciones son habitualmente usadas en distintas áreas como la Física, Matemática, Físico-química, Biofísica, y más, con lo cual, deben conocerse a la perfección.

Cofunciones

Son las recíprocas de las funciones trigonométricas básicas

recíproca del seno --> cosec α = r/y

recíproca del coseno --> sec α = r/x

recíproca de la tangente --> ctg α = x/y

Sabiendo el signo del seno y coseno, se deducen las recíprocas automáticamente.
